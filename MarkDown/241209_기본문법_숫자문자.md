## 문자열 포맷팅
``` python
>>> "오늘은 %s입니다" %"월요일"
'오늘은 월요일입니다'
```

``` python
>>> "오늘은 %d-%d-%02d입니다" %(24,12,9)
'오늘은 24-12-09입니다'
```

``` python
>>> "%d%%, %d%% 할인 쿠폰" %(per1, per2)
'10%, 20% 할인 쿠폰'
```

``` python
>>> "%10s" % "hello?"
'    hello?'
>>> "%-10s" % "hello?"
'hello?    '


```


``` python
>>> "{:<10}".format("hello?")
'hello?    '
>>> "{:>10}".format("hello?")
'    hello?'
>>> "{:^10}".format("hello?")
'  hello?  '
>>> "{:=^10}".format("hello?")
'==hello?=='

```





``` python
>>> "%f" % 3.1415
'3.141500'

>>> "%0.2f" % 3.1415
'3.14'
```

``` python
>>> "{0} {1} {2}".format("학교", "종이", "땡땡땡")
'학교 종이 땡땡땡'
>>> "{1} {2} {0}".format("학교", "종이", "땡땡땡")
'종이 땡땡땡 학교'
```
```python
>>> "{0}, {b}".format(10, b=11)
'10, 11'
```
