# PEP 8

PEP 8은 Python Enhancement Proposal 8의 약자로, 파이썬 코드 스타일 가이드를 정의한 문서입니다. PEP 8은 가독성이 높고 일관된 파이썬 코드 작성을 목표로 하며, 파이썬 코드를 작성할 때 따라야 할 코딩 규칙과 모범 사례를 제공.

[PEP8 바로가기](https://peps.python.org/pep-0008/)
