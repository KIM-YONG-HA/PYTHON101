## 37. 삽입 정렬(Insertion Sort)
* 손 안의 카드 정렬 방법 
* 두 번째 요소부터 시작해서 왼쪽의 모든 요소와 비교하며 적절한 위치를 찾음
* 필요하면 왼쪽 요소들을 한 칸씩 이동한 후 삽입
* 모든 요소에 대해 반복하여 정렬 완료
* 최고 n, 최악 n^2

```
초기 자료 : 8, 3, 4, 9, 7
```

```
[0]           => 8, 3, 4, 9, 7
[1]  3, 8 비교 => 3, 8, 4, 9, 7
[2]  4, 8 비교 => 3, 4, 8, 9, 7
[3]  9, 8 비교 => 3, 4, 8, 9, 7
[4]  7, 9 비교 => 3, 4, 8, 9, 7
[4]  7, 8 비교 => 3, 4, 8, 7, 9
[5]            => 3, 4, 7, 8, 9


```
