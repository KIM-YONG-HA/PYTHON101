## 63. c언어 | 연산자 


``` c
#include <stdio.h>
int main(int argc, char *argv[]){

    int a = 4;
    int b = 7;
    int c = 4 | 7;

    printf("%d", c);
    return 0;

}

```

### | 비트 연산자 OR

```
4 0000 0100
7 0000 0111
  0000 0111 // 둘 중에 하나라도 1이면 1
결과는 7 
```


### 주요 비트 연산자 

#### 5 & 3 (AND)
두 비트 모두 1이면 1
```
0000 0101 & 0000 0011 = 0000 0001(1)
```

#### 5 ^ 3 (XOR)
서로 다르면 1 같으면 0 
```
0000 0101 & 0000 0011 = 0000 0110(6)
```

### ~5(NOT)
```
~(0000 0101) = 1111 1010(-6)
```
